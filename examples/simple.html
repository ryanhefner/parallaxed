Parallaxed

Easily apply parallax effects to page elements as you scroll.

Selectors

Pass in a string, or array, of the selectors you’d like to target

Enter / Exit

Setup callbacks for when an element enters or exits the viewport.

Progress

Progress information is also supplied in case there are styles that need to be set based on the elements position within/through the viewport.

Resize

In addition to scroll events triggering the callbacks, resize events can trigger the callbacks as well.

Simple

There could be a lot of other default stuff that this could do, but I was trying to keep this simple.
Let me know if there’s something else you’d like to see this do, @ryanhefner.
